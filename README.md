## 项目列表

### 🐍 贪吃蛇游戏

经典的像素复古风格贪吃蛇游戏。

- **位置**: [`snake/`](snake/)
- **技术栈**: HTML5 Canvas, CSS3, Vanilla JavaScript
- **状态**: ✅ 已完成

```bash
# 快速启动
cd snake/
python3 -m http.server 8000
# 访问 http://localhost:8000
```

**功能特性**:
- 三种难度选择（简单/中等/困难）
- 实时计分和最高分持久化
- 暂停/继续功能
- 键盘控制（方向键或 WASD）

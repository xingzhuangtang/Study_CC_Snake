# 🐍 贪吃蛇 - 像素复古版

经典的贪吃蛇游戏，采用像素复古风格设计。

## 🎮 游戏特性

- **像素复古风格** - Press Start 2P 字体，经典配色
- **计分系统** - 实时计分，自动保存最高分
- **三种难度** - 简单、中等、困难
- **游戏控制** - 暂停/继续，难度选择
- **键盘操作** - 方向键或 WASD 控制

## 🚀 快速开始

直接用浏览器打开 `index.html` 即可开始游戏。

```bash
# 或者使用本地服务器
python3 -m http.server 8000
# 访问 http://localhost:8000
```

## 🎯 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制方向 |
| W A S D | 控制方向 |
| 空格 | 暂停/继续 |

## 📝 游戏规则

1. 使用方向键或 WASD 控制蛇的移动方向
2. 吃到食物得 10 分，蛇身变长
3. 撞到墙壁或自己，游戏结束
4. 随着难度提升，蛇的移动速度会加快

## 🏆 最高分

最高分保存在浏览器 localStorage 中，关闭浏览器后不会丢失。
